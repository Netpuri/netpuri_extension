# Chrome Extension WebCleanser

웹 콘텐츠의 텍스트를 필터링하고, 링크를 검사하여 사이트의 안정성을 확인하는 확장 프로그램입니다.

## How to Packing

확장프로그램을 빌드하는 방법입니다.

```
$ NODE_ENV=production npm run build
```
